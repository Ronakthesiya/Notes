## Create

```html

<div id="dn">
</div>

<script>
    $("#dn").dxNumberBox();
</script>

```

## Methods

| #  | Method                           | Parameters           | Return Type          | Description                                                                                                                                        | Example                                                                                            |
| -- | -------------------------------- | -------------------- | -------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| 1  | beginUpdate()                    | None                 | void                 | Prevents UI rendering while multiple changes are being made. Used with `endUpdate()` to improve performance when updating several options at once. | `numberBox.beginUpdate();`                                                                         |
| 2  | blur()                           | None                 | void                 | Removes focus from the number input field.                                                                                                         | `$("#box").dxNumberBox("blur");`                                                                   |
| 3  | clear()                          | None                 | void                 | Resets the value to the default value (usually null).                                                                                              | `$("#box").dxNumberBox("clear");`                                                                  |
| 4  | defaultOptions(rule)             | Object rule          | void                 | Static method used to define default configuration for all `dxNumberBox` instances depending on device type.                                       | `DevExpress.ui.dxNumberBox.defaultOptions({ device:{deviceType:"desktop"}, options:{width:200}});` |
| 5  | dispose()                        | None                 | void                 | Releases resources used by the component and removes event handlers. Should be called before removing the DOM element.                             | `$("#box").dxNumberBox("dispose");`                                                                |
| 6  | element()                        | None                 | HTMLElement / jQuery | Returns the root DOM element of the component.                                                                                                     | `var el = $("#box").dxNumberBox("element");`                                                       |
| 7  | endUpdate()                      | None                 | void                 | Applies all UI updates after `beginUpdate()` was called.                                                                                           | `numberBox.endUpdate();`                                                                           |
| 8  | focus()                          | None                 | void                 | Sets focus to the input element of the number box.                                                                                                 | `$("#box").dxNumberBox("focus");`                                                                  |
| 9  | getButton(name)                  | String name          | Button instance      | Returns a custom button instance added to the editor.                                                                                              | `var btn = $("#box").dxNumberBox("getButton","myBtn");`                                            |
| 10 | getInstance(element)             | DOM / jQuery element | NumberBox instance   | Static method used to obtain the component instance from a DOM element.                                                                            | `var inst = DevExpress.ui.dxNumberBox.getInstance(document.getElementById("box"));`                |
| 11 | instance()                       | None                 | NumberBox            | Returns the current NumberBox instance so other methods can be accessed.                                                                           | `var inst = $("#box").dxNumberBox("instance");`                                                    |
| 12 | off(eventName)                   | String eventName     | NumberBox            | Removes all event handlers attached to the specified event.                                                                                        | `numberBox.off("valueChanged");`                                                                   |
| 13 | off(eventName, handler)          | String, Function     | NumberBox            | Removes a specific event handler.                                                                                                                  | `numberBox.off("valueChanged", handler);`                                                          |
| 14 | on(eventName, handler)           | String, Function     | NumberBox            | Attaches an event handler to a component event.                                                                                                    | `numberBox.on("valueChanged", function(e){ console.log(e.value); });`                              |
| 15 | on(eventsObject)                 | Object               | NumberBox            | Subscribes to multiple events at once.                                                                                                             | `numberBox.on({ valueChanged:e=>{}, focusIn:e=>{} });`                                             |
| 16 | option()                         | None                 | Object               | Returns all configuration options of the component.                                                                                                | `var opts = numberBox.option();`                                                                   |
| 17 | option(name)                     | String               | Any                  | Returns the value of a specific option.                                                                                                            | `var val = numberBox.option("value");`                                                             |
| 18 | option(name,value)               | String, Any          | void                 | Updates a specific option dynamically.                                                                                                             | `numberBox.option("value",50);`                                                                    |
| 19 | option(options)                  | Object               | void                 | Updates multiple options at once.                                                                                                                  | `numberBox.option({ value:20, min:0 });`                                                           |
| 20 | registerKeyHandler(key, handler) | String key, Function | void                 | Registers custom behavior for keyboard keys.                                                                                                       | `numberBox.registerKeyHandler("enter", function(e){ alert("Pressed Enter"); });`                   |
| 21 | repaint()                        | None                 | void                 | Re-renders the component without reloading data. Useful when layout changes dynamically.                                                           | `numberBox.repaint();`                                                                             |
| 22 | reset(value)                     | Number value         | void                 | Resets the component value to the specified value and sets `isDirty` to false.                                                                     | `numberBox.reset(10);`                                                                             |



## Events

| #  | Event           | Type     | Default | Trigger Condition                                      | Description                                                                                                 | Example                                                  |
| -- | --------------- | -------- | ------- | ------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| 1  | onChange        | Function | null    | When input loses focus after value changed by keyboard | Triggered when the user modifies the value and the editor loses focus.                                      | `onChange: function(e){ console.log("Changed"); }`       |
| 2  | onContentReady  | Function | null    | After component rendering                              | Fired when the NumberBox is rendered or repainted. Useful for DOM manipulation after load.                  | `onContentReady: function(e){ console.log("Ready"); }`   |
| 3  | onCopy          | Function | null    | When text is copied from input                         | Executes when the user copies content from the input field.                                                 | `onCopy: function(e){ console.log("Copied"); }`          |
| 4  | onCut           | Function | null    | When text is cut from input                            | Runs when the user cuts text from the number box.                                                           | `onCut: function(e){ console.log("Cut event"); }`        |
| 5  | onDisposing     | Function | null    | Before component destruction                           | Fired before the component instance is disposed and removed from DOM.                                       | `onDisposing: function(e){ console.log("Disposed"); }`   |
| 6  | onEnterKey      | Function | null    | When Enter key pressed while focused                   | Triggered when the Enter key is pressed inside the number box.                                              | `onEnterKey: function(e){ alert("Enter pressed"); }`     |

| 10 | onInput         | Function | null    | On every input change while focused                    | Fired whenever the user types or modifies the input while focused.                                          | `onInput: function(e){ console.log("Typing..."); }`      |

| 13 | onOptionChanged | Function | null    | When any option value changes                          | Executes when a component configuration option is modified dynamically.                                     | `onOptionChanged: function(e){ console.log(e.name); }`   |
| 14 | onPaste         | Function | null    | When content is pasted                                 | Fired when the user pastes content into the input.                                                          | `onPaste: function(e){ console.log("Pasted"); }`         |
| 15 | onValueChanged  | Function | null    | When component value changes                           | Executes when the value changes either by user input or programmatically. Provides new and previous values. | `onValueChanged: function(e){ console.log(e.value); }`   |
