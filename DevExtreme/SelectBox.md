## Create

```js

$(function() {
    $("#selectBox").dxSelectBox({ });
});

```

## Methods

| #  | Method                           | Parameters       | Return Type          | Description                                                                                                     | Example                                            |
| -- | -------------------------------- | ---------------- | -------------------- | --------------------------------------------------------------------------------------------------------------- | -------------------------------------------------- |
| 1  | beginUpdate()                    | none             | void                 | Stops UI rendering until `endUpdate()` is called. Useful when updating multiple options to improve performance. | `selectBox.beginUpdate();`                         |
| 2  | endUpdate()                      | none             | void                 | Resumes rendering after `beginUpdate()` and applies all changes.                                                | `selectBox.endUpdate();`                           |
| 3  | blur()                           | none             | void                 | Removes focus from the SelectBox input element.                                                                 | `$("#sb").dxSelectBox("blur");`                    |
| 4  | focus()                          | none             | void                 | Sets focus to the SelectBox input field.                                                                        | `$("#sb").dxSelectBox("focus");`                   |
| 5  | clear()                          | none             | void                 | Clears the selected value and resets it to default.                                                             | `$("#sb").dxSelectBox("clear");`                   |
| 6  | close()                          | none             | void                 | Closes the dropdown list if it is open.                                                                         | `$("#sb").dxSelectBox("close");`                   |
| 7  | open()                           | none             | void                 | Opens the dropdown list programmatically.                                                                       | `$("#sb").dxSelectBox("open");`                    |
| 8  | content()                        | none             | HTMLElement / jQuery | Returns the popup content element of the dropdown.                                                              | `var el = selectBox.content();`                    |
| 9  | defaultOptions(rule)             | Object           | void                 | Static method used to set default configuration for all SelectBox instances depending on device type.           | `DevExpress.ui.dxSelectBox.defaultOptions({...});` |
| 10 | dispose()                        | none             | void                 | Releases all resources used by the component and removes event handlers.                                        | `$("#sb").dxSelectBox("dispose");`                 |
| 11 | element()                        | none             | HTMLElement / jQuery | Returns the root DOM element of the SelectBox.                                                                  | `var el = selectBox.element();`                    |
| 12 | getButton(name)                  | String           | Button instance      | Returns an instance of a custom button added to the SelectBox editor.                                           | `var btn = selectBox.getButton("myButton");`       |
| 13 | getDataSource()                  | none             | DataSource           | Returns the component’s DataSource instance.                                                                    | `var ds = selectBox.getDataSource();`              |
| 14 | instance()                       | none             | SelectBox instance   | Returns the SelectBox instance object so you can call methods on it.                                            | `var inst = $("#sb").dxSelectBox("instance");`     |
| 15 | off(eventName)                   | String           | SelectBox            | Removes all event handlers attached to the specified event.                                                     | `selectBox.off("valueChanged");`                   |
| 16 | off(eventName, handler)          | String, Function | SelectBox            | Removes a specific event handler.                                                                               | `selectBox.off("valueChanged", handler);`          |
| 17 | on(eventName, handler)           | String, Function | SelectBox            | Subscribes to a specific event.                                                                                 | `selectBox.on("valueChanged", function(e){ });`    |
| 18 | on(eventsObject)                 | Object           | SelectBox            | Subscribes to multiple events at once.                                                                          | `selectBox.on({ opened:e=>{}, closed:e=>{} });`    |
| 19 | option()                         | none             | Object               | Returns all configuration options of the component.                                                             | `var opts = selectBox.option();`                   |
| 20 | option(name)                     | String           | Any                  | Returns value of a specific option.                                                                             | `var val = selectBox.option("value");`             |
| 21 | option(name,value)               | String, Any      | void                 | Updates a specific option dynamically.                                                                          | `selectBox.option("value", 2);`                    |
| 22 | option(options)                  | Object           | void                 | Updates multiple options at once.                                                                               | `selectBox.option({value:1, disabled:true});`      |
| 23 | registerKeyHandler(key, handler) | String, Function | void                 | Registers custom keyboard event handler for a specific key.                                                     | `selectBox.registerKeyHandler("enter", fn);`       |
| 24 | repaint()                        | none             | void                 | Re-renders the component UI. Useful after layout changes.                                                       | `selectBox.repaint();`                             |
| 25 | reset(value)                     | Any              | void                 | Resets the value to the specified value and clears dirty state.                                                 | `selectBox.reset(null);`                           |

| -- | -------------------- | -------- | ------- | ----------------------------------------- | ------------------------------------------------------------------------------------------------------ | --------------------------------------------------------- |
| 1  | onChange             | Function | null    | When input loses focus after value change | Triggered when the editor loses focus and the value was modified.                                      | `onChange: function(e){ console.log("Changed"); }`        |
| 2  | onClosed             | Function | null    | When dropdown closes                      | Fires after the dropdown list is closed.                                                               | `onClosed: function(){ console.log("Dropdown closed"); }` |
| 3  | onContentReady       | Function | null    | After component rendering                 | Executed after the component is rendered or repainted.                                                 | `onContentReady: function(){ console.log("Ready"); }`     |

| 6  | onCut                | Function | null    | When text is cut                          | Executes when text is cut from the input.                                                              | `onCut: function(){}`                                     |

| 15 | onOpened             | Function | null    | When dropdown opens                       | Fired after the dropdown list is opened. ([DevExtreme][2])                                             | `onOpened: function(){}`                                  |
| 16 | onOptionChanged      | Function | null    | When component option changes             | Triggered whenever a property of the component changes dynamically. ([DevExtreme][2])                  | `onOptionChanged: function(e){}`                          |

| 18 | onSelectionChanged   | Function | null    | When selected item changes                | Fired when the user selects a different item from the dropdown list.                                   | `onSelectionChanged: function(e){}`                       |
