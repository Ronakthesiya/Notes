## Create DateBox

```html
<div id="dateBox">
</div>

<script>
    $("#dateBox").dxDateBox();
</script>    
```

## Methods

| #  | Method                             | Description                                                                                                                                  | Syntax Example                                        | Practical Use Case                                                |
| -- | ---------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------- | ----------------------------------------------------------------- |
| 1  | `beginUpdate()`                    | Temporarily stops component rendering to improve performance when updating multiple options. Rendering resumes when `endUpdate()` is called. | `dateBox.beginUpdate();`                              | Use when updating many properties to avoid multiple UI refreshes. |
| 2  | `endUpdate()`                      | Resumes rendering after `beginUpdate()`.                                                                                                     | `dateBox.endUpdate();`                                | Finalize batch updates.                                           |
| 3  | `blur()`                           | Removes focus from the DateBox input element.                                                                                                | `dateBox.blur();`                                     | Remove cursor focus from the field programmatically.              |
| 4  | `focus()`                          | Sets focus to the DateBox input field.                                                                                                       | `dateBox.focus();`                                    | Automatically focus the date field when a form loads.             |
| 5  | `clear()`                          | Clears the current value and resets it to default (`null`).                                                                                  | `dateBox.clear();`                                    | Clear selected date when user clicks reset button.                |
| 6  | `open()`                           | Opens the date/time picker dropdown.                                                                                                         | `dateBox.open();`                                     | Open calendar programmatically.                                   |
| 7  | `close()`                          | Closes the dropdown picker.                                                                                                                  | `dateBox.close();`                                    | Close calendar after selecting date.                              |
| 8  | `content()`                        | Returns the popup’s content element (calendar UI container).                                                                                 | `var popupContent = dateBox.content();`               | Customize popup or access calendar DOM.                           |
| 9  | `field()`                          | Returns the `<input>` HTML element of the DateBox.                                                                                           | `var input = dateBox.field();`                        | Modify input attributes dynamically.                              |
| 10 | `element()`                        | Returns the root DOM element of the DateBox component.                                                                                       | `dateBox.element();`                                  | Access container element for styling or manipulation.             |
| 11 | `dispose()`                        | Releases resources used by the component and removes event handlers.                                                                         | `$("#dateBox").dxDateBox("dispose");`                 | Destroy component when removing it from DOM.                      |
| 12 | `instance()`                       | Returns the current DateBox instance.                                                                                                        | `var instance = $("#dateBox").dxDateBox("instance");` | Required to call methods programmatically.                        |
| 13 | `getInstance(element)`             | Static method that gets the DateBox instance from a DOM element.                                                                             | `DevExpress.ui.dxDateBox.getInstance(el);`            | Useful when component was created elsewhere.                      |
| 14 | `getButton(name)`                  | Returns an instance of a custom button added to the DateBox.                                                                                 | `dateBox.getButton("myButton")`                       | Access or control custom action buttons.                          |
| 15 | `option()`                         | Gets all component options.                                                                                                                  | `dateBox.option();`                                   | Inspect current configuration.                                    |
| 16 | `option(name)`                     | Gets a specific option value.                                                                                                                | `dateBox.option("value");`                            | Retrieve selected date.                                           |
| 17 | `option(name, value)`              | Sets a specific option value.                                                                                                                | `dateBox.option("value", new Date());`                | Update date programmatically.                                     |
| 18 | `option(options)`                  | Updates multiple options at once.                                                                                                            | `dateBox.option({ disabled:true, width:200 });`       | Change several properties simultaneously.                         |
| 19 | `on(eventName, handler)`           | Attaches an event handler.                                                                                                                   | `dateBox.on("valueChanged", function(e){})`           | Listen for date changes dynamically.                              |
| 20 | `on(events)`                       | Attaches multiple event handlers at once.                                                                                                    | `dateBox.on({ valueChanged: fn1, opened: fn2 });`     | Handle several events in one call.                                |
| 21 | `off(eventName)`                   | Removes all handlers from a specific event.                                                                                                  | `dateBox.off("valueChanged");`                        | Stop listening to an event.                                       |
| 22 | `off(eventName, handler)`          | Removes a specific event handler.                                                                                                            | `dateBox.off("valueChanged", handler);`               | Remove a particular callback.                                     |
| 23 | `registerKeyHandler(key, handler)` | Registers a keyboard event handler for specific keys.                                                                                        | `dateBox.registerKeyHandler("enter", fn);`            | Perform action when user presses Enter.                           |
| 24 | `repaint()`                        | Re-renders the component UI without reloading data.                                                                                          | `dateBox.repaint();`                                  | Refresh UI after dynamic style changes.                           |
| 25 | `defaultOptions(rule)`             | Static method that sets default configuration for all DateBox components based on device type.                                               | `DevExpress.ui.dxDateBox.defaultOptions({...})`       | Apply global settings for desktop/mobile.                         |
